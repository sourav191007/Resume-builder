# ResumeCraft – Resume Builder

ResumeCraft is a responsive web application that helps users create professional resumes quickly through an intuitive form-based interface. The application provides a live preview of the resume as users enter their information and allows them to generate a clean, printable resume.

## Features

* Live resume preview
* Responsive and user-friendly interface
* Add personal information, education, skills, projects, and work experience
* Dynamic updates without refreshing the page
* Print and download resume as PDF
* Clean and professional resume template

## Tech Stack

**Frontend**

* HTML5
* CSS3
* JavaScript

**Backend**

* Node.js
* Express.js
* EJS

## Installation

1. Clone the repository:

   ```bash
   git clone https://github.com/your-username/resumecraft.git
   ```

2. Navigate to the project directory:

   ```bash
   cd resumecraft
   ```

3. Install dependencies:

   ```bash
   npm install
   ```

4. Start the server:

   ```bash
   node index.js
   ```

   or

   ```bash
   nodemon index.js
   ```

5. Open your browser and visit:

   ```
   http://localhost:3000
   ```
